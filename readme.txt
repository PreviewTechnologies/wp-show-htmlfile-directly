## Description

This `WP Show HTML Page Directly` plugin will let you map a html file location for any specific post or page to display directly HTML page for that post or page.

For more details you can send mail with your suggestions, recommendation to shaharia.azam@gmail.com


## Installation ==

To **Install** this `WP Show HTML Page Directly` plugin there is no any complexity. It's very simple like other plugin. Just follow the procedure described below.

1. Download the plugin from Wordpress Plugin repository or GitHub repository. After downloading the zip file extract it.
2. Upload `wp-show-htmlfile-directly.zip` plugin directory to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. From your specific post or page add a custom field with name `display_html_file` and in value field provide the html file path which you want to display for this post
